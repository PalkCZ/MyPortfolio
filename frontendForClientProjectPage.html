Frontend For Client Project

Project Type Professional / Solo Project
Software Used Unreal Engine 4
Languages Used C++ / Blueprints
Primary Role(s) Everything

This was my first professional project in Unreal Engine 4. It was a freelance work I got on Upwork. The job was to create good looking UI in Unreal Engine 4.

I got a design to take a reference from in a Figma project client showed me. But because it was just a reference I had to design the UI mostly on my own with client giving the feedback. After a few tries we agreed on UI he liked and I implemented that.

The following images show the UI I designed in Unreal Engine 4. The buttons have hover effects which are shown too.






The implementation after designing the UI was pretty straightforward and easy because I was supposed to only make the frontend. So all the implementation I had to do was to make sure that the UI is responsive and works well.

The following code snippet shows how opening registration widget works. All opening functions work mostly the same way but with different widgets. 

//this function gets called when user tries to reach the register menu by clicking on the appropriate button
	void AMyHUD::OpenRegistrationWidget() {
		//user can reach the register menu only from login menu
		//it's important to close the login menu so it doesn't block the register menu
		if (loginWidget && loginWidget->GetIsVisible()) {
			loginWidget->RemoveFromViewport();
		}
		//creates the register menu
		registrationWidget = CreateWidget<UMyRegistrationUserWidget>(GetWorld(), registrationWidgetTemplete);
		registrationWidget->AddToViewport();
	}
