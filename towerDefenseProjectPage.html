Tower Defense Project

Project Type 2 people project
Software Used Unity
Languages Used C#
Primary Role(s) /Programmer

This project was done as a collaboration with a graphic artist to create something in Unity where we could both showcase our skills. The game is a typical tower defense game. All the game code was done by me. The following sections contain snippets of code but not all of it. If you want to check out all of it you can do so here.

I got a design to take a reference from in a Figma project client showed me. But because it was just a reference I had to design the UI mostly on my own with client giving the feedback. After a few tries we agreed on UI he liked and I implemented that.

The following images show the UI I designed in Unreal Engine 4. The buttons have hover effects which are shown too.






The implementation after designing the UI was pretty straightforward and easy because I was supposed to only make the frontend. So all the implementation I had to do was to make sure that the UI is responsive and works well.

The following code snippet shows that camera movement is done by rotation of the camera. The only difference between horizontal and vertical is the axis they affect. InvertedMouse can be only 1 or -1 based on if the mouse is set as inverted or not. 

void AMyPlayer::CameraHorizontal(float horizontal) {
	if (!lockedOnEnemy) {
		cameraAttach->AddLocalRotation(FRotator(0, cameraRotationSpeed * horizontal, 0));
	}
}

void AMyPlayer::CameraVertical(float vertical) {
	if (!lockedOnEnemy) {
		cameraAttach->AddLocalRotation(FRotator(cameraRotationSpeed * vertical * invertedMouse, 0, 0));
	}
}
